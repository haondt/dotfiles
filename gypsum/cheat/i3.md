# i3

## mod+enter
open new terminal

## mod+shift+arrow keys
move window

## mod+f
toggle fullscreen

## mod+shift-space
toggle floating window

## mod+[0-9]
#desktop

go to workspace

## mod+shift+[0-9]
#desktop

move window to workspace

## mod+s
#run

open application launcher (rofi)

## mod+shift+c
reload configuration file

## mod+shift+r
restart i3 inplace

## mod+shift+e
exit i3

## mod+shift+minus
#shadow-realm

move window to scratchpad

## mod+minus
#shadow-realm

toggle through windows in scratchpad

## mod+tab
show window switcher

## mod+shift+space
#scratchpad #shadow-realm

toggle window float

## mod+r
toggle resize mode

## escape | return | mod+r
exit resize mode

## mod+q
close focused window
