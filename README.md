# dotfiles
Cause I've reinstalled linux too many damn times

This repo contains my dotfiles and other configuration. It has evolved over the years and for big changes (different OSes) I usually just outright add a new folder. Many of them are in use in some way or another, but at the time of writing, [kunzite](./kunzite) is the most actively developed.

### Configs

- [Aurora](./Auroroa) (deprecated): Config for a basic Manjaro i3 setup with zsh and vim.
- [Shale](./Shale) (deprecated): Very basic config for Manjaro i3, vim and zsh.
- [Windows](./Windows): Some scripts for git-bash and VsVim.
- [gypsum](./gypsum) (maintenance mode): Large config collection for various tools and colorschemes, mostly targeted at Debian i3 and Debian WSL with Neovim.
- [kunzite](./kunzite): Evolution of gypsum, targeting Fedora Sway and Fedora WSL. Uses ansible for templating and deploying configuration.
